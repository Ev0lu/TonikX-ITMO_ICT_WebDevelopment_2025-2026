# Задание 5 — HTTP сервер «Оценки»

## Цель
Обработать HTML‑форму (POST), провалидировать параметры и отобразить список оценок.

## Протокол
HTTP/1.1 поверх TCP. Ответы с корректными заголовками и редиректами.

## Файлы
- `Lr1/ex5/server.py`
- `Lr1/ex5/template.html`

## Логика сервера
1. GET `/` — отдать HTML, подставив текущие оценки в шаблон.
2. POST `/` — распарсить `application/x-www-form-urlencoded`, валидировать и сохранить.
3. После успешного добавления выполнить редирект `302` на `/`.

### Валидация
- Дисциплина: не пуста, длина ≤ 100, допустимые символы `[а-яА-Яa-zA-Z0-9\\s\\-\\.]`.
- Оценка: целое число 1..5.

### Хранение данных
- В памяти процесса: `self.grades = {}`. При перезапуске сервера очищается.
- Для персистентности можно добавить запись/чтение JSON.

## Запуск
`python server.py <host> <port>`, например: `python server.py localhost 8080`  
Открыть `http://localhost:8080/`

## Нюансы
- Тело POST берется после разделителя `\\r\\n\\r\\n`.
- Для корректной длины ответа используется `Content-Length` по байтам UTF‑8.
