# Задание 3 — Мини‑HTTP сервер

## Цель
Сформировать корректный HTTP/1.1 ответ и отдать статический HTML.

## Протокол
HTTP/1.1 поверх TCP. Обязательны строка статуса и заголовки.

## Файлы
- `Lr1/ex3/server.py`
- `Lr1/ex3/Index.html` (статическая страница)

## Логика сервера
1. `listen()` и `accept()` входящие соединения.
2. Прочитать первый блок запроса (достаточно стартовой строки и заголовков).
3. Если HTML существует — вернуть `200 OK`, иначе `404 Not Found`.
4. Заголовки минимум:
   - `Content-Type: text/html; charset=utf-8`
   - `Content-Length: <длина_в_байтах>`
   - `Connection: close`

## Запуск
`cd ...\Lr1\ex3 && python server.py`, затем открыть `http://localhost:8080/`.
