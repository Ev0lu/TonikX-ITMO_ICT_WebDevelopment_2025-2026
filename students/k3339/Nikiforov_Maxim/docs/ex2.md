# Задание 2 — TCP калькулятор

## Цель
Научиться принимать TCP‑подключения и обрабатывать простые текстовые команды.

## Протокол и API
- Адресное семейство: `AF_INET` (IPv4)
- Тип сокета: `SOCK_STREAM` (TCP)
- Ключевые вызовы:
  - Сервер: `socket()`, `bind()`, `listen()`, `accept()`, `recv()`, `sendall()`, `close()`
  - Клиент: `socket()`, `connect()`, `sendall()`, `recv()`, `close()`

## Файлы
- `Lr1/ex2/server.py`
- `Lr1/ex2/client.py`

## Команды
- `PYTH a b` — гипотенуза: `sqrt(a^2 + b^2)`
- `QUAD a b c` — решение квадратного уравнения `ax^2 + bx + c = 0`
- `TRAP a b h` — площадь трапеции `(a + b)/2 * h`
- `PARA a h` — площадь параллелограмма `a * h`

Разбор аргументов: строка запроса разбивается на части, `parts[0]` — операция, `parts[1:1+n]` — ровно `n` числовых аргументов.

## Запуск
1. Сервер: `cd ...\Lr1\ex2 && python server.py`
2. Клиент: `cd ...\Lr1\ex2 && python client.py`, затем вводить команды.

## Нюансы TCP
- Поток байт без сохранения границ сообщений — определять конец команды нужно самим (например, перевод строки).
- Доставка и порядок гарантированы, соединение устанавливается перед обменом.
